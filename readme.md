# Inventory Management System

## Features

- Product Management
- Customer Management
- Order Management
- Inventory Validation
- Automatic Stock Reduction
- Unique SKU Validation
- Unique Email Validation

## Tech Stack

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy

Frontend:
- React
- Axios
- React Router

## Run Backend

cd backend

python -m uvicorn main:app --reload

## Run Frontend

cd frontend

npm install

npm run dev
